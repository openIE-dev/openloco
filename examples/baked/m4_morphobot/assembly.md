# Mechanical assembly

Root link(s): `base`

1. Attach **FL_roll_housing** to **base** via joint `FL_HIP_roll` (revolute).
2. Attach **FR_roll_housing** to **base** via joint `FR_HIP_roll` (revolute).
3. Attach **RL_roll_housing** to **base** via joint `RL_HIP_roll` (revolute).
4. Attach **RR_roll_housing** to **base** via joint `RR_HIP_roll` (revolute).
5. Attach **FL_leg** to **FL_roll_housing** via joint `FL_HIP_pitch` (revolute).
6. Attach **FR_leg** to **FR_roll_housing** via joint `FR_HIP_pitch` (revolute).
7. Attach **RL_leg** to **RL_roll_housing** via joint `RL_HIP_pitch` (revolute).
8. Attach **RR_leg** to **RR_roll_housing** via joint `RR_HIP_pitch` (revolute).
9. Attach **FL_rotor** to **FL_leg** via joint `FL_ROTOR_drive` (continuous).
10. Attach **FR_rotor** to **FR_leg** via joint `FR_ROTOR_drive` (continuous).
11. Attach **RL_rotor** to **RL_leg** via joint `RL_ROTOR_drive` (continuous).
12. Attach **RR_rotor** to **RR_leg** via joint `RR_ROTOR_drive` (continuous).
