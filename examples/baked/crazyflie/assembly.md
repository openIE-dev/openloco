# Mechanical assembly

Root link(s): `base`

1. Attach **FL_motor** to **base** via joint `FL_motor_mount` (fixed).
2. Attach **FR_motor** to **base** via joint `FR_motor_mount` (fixed).
3. Attach **RL_motor** to **base** via joint `RL_motor_mount` (fixed).
4. Attach **RR_motor** to **base** via joint `RR_motor_mount` (fixed).
5. Attach **FL_prop** to **FL_motor** via joint `FL_PROP_drive` (continuous).
6. Attach **FR_prop** to **FR_motor** via joint `FR_PROP_drive` (continuous).
7. Attach **RL_prop** to **RL_motor** via joint `RL_PROP_drive` (continuous).
8. Attach **RR_prop** to **RR_motor** via joint `RR_PROP_drive` (continuous).
